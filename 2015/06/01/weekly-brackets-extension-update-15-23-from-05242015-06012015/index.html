Weekly Brackets Extension Update - 15/23

Filed Under Update

Welcome to This Week’s Brackets Extension Updates, Issue 15/23.

Added or updated @Brackets extensions in the week from 05/25/2015 to 06/01/2015.

Last week there were 5 new Extensions and 7 updates. This includes themes and other extensions.

What’s going on?

I’m late, late, late and still catching up with the previous weeks. Too much things going on at the same time. New projects and maintenance of old projects :-)

One important update: there is a new way to get in contact with the team. We have created a Slack Channel where you can ask questions about Brackets and talk about features.

Have a great week!

In the meantime, you can find us on these social networks

  • Facebook
  • Google Plus
  • twitter
  • Slack Channel

Share your thoughts and stories, ask questions and get in touch with the team.

As usual, you can download the latest Brackets release. Please let us know what you think.

Brackets Script Builder and framework launcher 0.1.0 Allows to run programs contained in one file (can be used for Scala, Java, Python, Ruby, Node, C++, Bash) from Brackets and display results in panel. It is possible to create own build systems via ‘Edit > Script Builder Configuration’ menu item and editing opened JSON file (you need to restart Brackets afterwards). Based on Brackets Builder (http://github.com/Vhornets/brackets-builder). I add to the original one a framework management and also an OS detection for execution. Add frameworks over the UI Keyboard shortcuts: Alt-F12 to show the dialog for framework choice / add. Alt-F8 to launch the environment.
Alt-F1 to run current file as a script. F10 to compile current file. * F11 to run compilation result.
