Updated Extensions

Name Version Description Download
Display Shortcuts 1.3.4 Display current shortcuts in a bottom panel that can be sorted and filtered. Add and disable shortcuts from context menu.
Autoprefixer 0.6.0 Parse CSS and add vendor prefixes automatically.
Brackets Builder 0.0.5 Allow to run build programs (such as running Python/Ruby/Node/etc scripts) from Brackets and display results in panel. It is possible to create own build systems via ‘Edit>Edit Builder’ menu item and editing opened JSON-file (you need to restart Brackets). Press Ctrl(Cmd)-B to build current file.
Cardboard 0.4.0 Manage packages/dependencies for your project.
White Space Sanitizer 1.1.1 Bring sanity to your white spaces and tabs leveraging Brackets settings
Monokai Dark Soda 1.0.6 Dark theme based on Dark Soda and Monokai color schemes.
Ant Theme 1.0.6 Dark color theme meant to be easy on the eyes
Quick Dark 1.1.3 A dark theme (based on Easy Dark) that makes your code pretty and readable.
Quick Light 1.1.3 A light theme that makes your code pretty and readable.
Projects 0.1.7 Open your projects from handy modal dialog. Really helpful if you have a lot of projects.
CoffeeScript 1.1.7 Improves syntax highlighting for CoffeeScript and CJSX.
Show Recently Used Items 0.5.7 Add a menu at the top of the editor with a list of all the files shown recently
Brackets Additional Right Click Menu 0.5.1 A Brackets extension that add additional functionality to Brackets editor right menu, like cut, copy, paste, select all, Camel Case, UPPERCASE, lowercase, fix indentation and some functionality from edit menu. changelog: added a shortcuts to some menu items, Fixed some bugs with brackets 1.3 paste, added fix indentation, bug fix for uppercase, lowercase and camelcase selection behaviour and change it’s short cut key
Tab Out 1.0.2 Use the Tab key to jump out of nested elements surrounded by characters “, ‘, ), ] or } like in Eclipse.
Vintage Terminal Effect 1.1.0 Enhance your editor area with visual effects of Vintage Terminal. Supporting any dark theme for Brackets.
Select Word 1.0.1 Select the word at the current cursor postion like if you double-click there.
Brackets Script Builder 0.0.8 Allows to run programs contained in one file (can be used for Scala, Java, Python, Ruby, Node, C++, Bash) from Brackets and display results in panel.
It is possible to create own build systems via ‘Edit > Script Builder Configuration’ menu item and editing opened JSON file (you need to restart Brackets afterwards).
Based on Brackets Builder (http://github.com/Vhornets/brackets-builder).

Keyboard shortcuts:
F9 to run current file as a script.
 F10 to compile current file.
* F11 to run compilation result.
