Weekly Brackets Extension Update - 15/22

Filed Under Update

Welcome to This Week’s Brackets Extension Updates, Issue 15/22.

Added or updated @Brackets extensions in the week from 05/18/2015 to 05/25/2015.

Last week there were 8 new Extensions and 16 updates. This includes themes and other extensions.

What’s going on?

I’m late, late, late and still catching up with the previous weeks. Too much things going on at the same time. New projects and maintenance of old projects :-)

One important update: there is a new way to get in contact with the team. We have created a Slack Channel where you can ask questions about Brackets and talk about features.

Have a great week!

In the meantime, you can find us on these social networks

  • Facebook
  • Google Plus
  • twitter
  • Slack Channel

Share your thoughts and stories, ask questions and get in touch with the team.

As usual, you can download the latest Brackets release. Please let us know what you think.

New Extensions

Name Version Description Download
Lucee 0.1.1 Lucee Support for Brackets.
Bộ gõ Tiếng Việt Mudim 0.9.0 Trên Linux khi gõ Tiếng Việt thông qua các bộ gõ như IBus, nó sẽ không trực tiếp nhúng ký tự ngay khi ta gõ. Chính vì thế chức năng autocomplete trong Brackets sẽ không hoạt động như mong đợi. Extension Bộ gõ Tiếng Việt Mudim sẽ giúp khắc phục vấn đề đó. Lưu ý: Tắt bộ gõ Tiếng Việt trên máy khi dùng extension này.
Bộ gõ Tiếng Việt AVIM 0.9.0 Hỗ trợ gõ Tiếng Việt trong Brackets, không bị lỗi chức năng Autocomplete khi dùng trên Linux. Lưu ý: Tắt bộ gõ Tiếng Việt trên máy (ví dụ: ibus-unikey) khi dùng extension này.
Sand 1.0.0 A desert inspired theme for Brackets
Neon Highway 1.0.0 A neon theme for Brackets
Rolando Zabala II’s Dark Theme 1.0.0 A dark theme based on Brackets Default Dark Theme template
DarkPanja 1.0.0 A Dark Theme By Panja

Updated Extensions

Name Version Description Download
Brackets Git 0.14.27 Integration of Git into Brackets
Autosave Files on Window Blur 1.0.6 Autosave all open files when switching applications, in the style of PHPStorm/WebStorm.
QuickDocsJS 1.6.6 Inline short documentation for JavaScript functions, including a summary,syntax and parameters. Docs for jQuery,NodeJS and some basic ReactJS functions.
Preferences setup UI 1.2.0 Form for editing preferences.
JSCS for Brackets 0.7.0 Adds JSCS support to Brackets. JSCS is a code style linter for programmatically enforcing your style guide. You can configure JSCS for your project in detail using over 120 validation rules.
ECO template syntax highlighting 1.0.2 Add syntax highlighting support for the Eco templating language (.eco).
Extensions Rating 0.7.0 Shows extension’s download count(daily, overall, average and per version) and GitHub stats(stars and forks). Find most usable and popular extensions for Brackets via badges system! Sort extensions and find themes!
Documents Toolbar 0.5.15 Adds toolbar with tabs of open documents on the top of the editor. Now with split view support!
Slack Snippet 0.3.1 Push code snippets to a slack channel
Brackets Editor Bookmarks 0.10.4 Bookmark lines in the editor as you work so you can quickly jump back to them later. Open the “Bookmarks” Panel to see all of the bookmarks and open them in the editor.
PHP SmartHints 1.2.0 Provides code hints for PHP files including PHP keywords, built-in functions, constants and variables. Also provides hinting for variables in the current open PHP document. Note that this is a continuation of the former [PHP-SIG] Brackets-PHP-SmartHints and not a new extension or fork. The Brackets Registry flags it as a new extension if you change the name (makes sense).
Geekowater 0.1.1 Tema Geeko basado en la guía de estilo de Peter Gabriel’s Debut Album en TheDaysColor.
Go to Matching Bracket 2.1.0 Instantly locate and place the cursor on the matching bracket to the one under the cursor. Simply place the cursor on any one of {, }, (, ), [ or ] then press CTRL-ALT-RIGHT (CMD-ALT-RIGHT on Mac) or use the ‘Navigate… Go to Matching Bracket’ menu item to locate and place the cursor on the matching opening or closing bracket. NOTE: v2.0.0 changed the hotkey for compatibility with Brackets 1.3.
Pop-up Menu Brackets 2.2.2 A Brackets extension that add additional functionality to Brackets editor right menu like cut, copy, paste, save all, Camel Case, UPPERCASE, lowercase and some functionality from edit menu.
UI too small 0.0.4 Increase font sizes for people working on high dpi displays on Linux
M4 Macro Syntax Highlighter 0.1.1 Simple Syntax highlighting for M4 Macro files (e.g. Autoconf). To improve editting of configure.ac files primarily.
