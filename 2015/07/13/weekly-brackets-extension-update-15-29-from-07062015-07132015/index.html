Weekly Brackets Extension Update - 15/29

Filed Under Update

Welcome to This Week’s Brackets Extension Updates, Issue 15/29.

Added or updated @Brackets extensions in the week from 07/06/2015 to 07/13/2015.

Last week there were 3 new Extensions and 11 updates. This includes themes and other extensions.

What’s going on?

I’m late, late, late and still catching up with the previous weeks. Too much things going on at the same time. New projects and maintenance of old projects :-)

One important update: there is a new way to get in contact with the team. We have created a Slack Channel where you can ask questions about Brackets and talk about features.

Have a great week!

In the meantime, you can find us on these social networks

  • Facebook
  • Google Plus
  • twitter
  • Slack Channel

Share your thoughts and stories, ask questions and get in touch with the team.

As usual, you can download the latest Brackets release. Please let us know what you think.

Updated Extensions

Name Version Description Download
Tab tags 1.0.9 Indents HTML/XML tags and JS & PHP parentheses.
colorHints 1.1.2 Show colors you used in your CSS file before as code hints.
Brackets Dash 2.1.3 Adds Dash integration with Brackets. Requires Dash and OSX
JSHint Configurator 1.1.6 Enables configuration of JSHint options. Reads from current document’s inline configuration directive or from ‘.jshintrc’ configuration file. Extract JSHint options to JSON format.
BluePrint Beta 0.9.3 Code minimap and hierarchic outline viewer for JavaScript, HTML, CSS, LESS, SCSS, PHP and python.
HTML Format 0.2.8 An extension containing menubar and keyboard shortcuts for inserting HTML tags. Currently the list of available HTML tags for inserting through the menubar or a keyboard shortcut mimics the current Dreamweaver options. A preference window is available to change the default keyboard shortcuts.
Funkemedia Dark Theme 1.0.1 A Dark Theme based on an inverted version of Dreamweavers light standard theme. Great if you are used to dreamweavers colours but a lot easier on the eye.
Ionic Framework Code Hinting 0.4.0 Code hinting for the Ionic Framework
PHP SmartHints 1.2.2 Provides code hints for PHP files including PHP keywords, built-in functions, constants and variables. Also provides hinting for variables in the current open PHP document. Note that this is a continuation of the former [PHP-SIG] Brackets-PHP-SmartHints and not a new extension or fork. The Brackets Registry flags it as a new extension if you change the name (makes sense).
Brackets SASS 2.0.3-132 IMPORTANT Read “More info…” if you have issues removing or installing. Compiles SASS .sass/.scss files when changed. Updates styles during Live Preview. Experimental support for Quick Edit.
Subtly Light 1.0.3 A subtly colored light theme. Designed for legibility, quick reference and comfort.
