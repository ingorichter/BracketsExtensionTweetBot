Weekly Brackets Extension Update - 15/32

Filed Under Update

Welcome to This Week’s Brackets Extension Updates, Issue 15/32.

Added or updated @Brackets extensions in the week from 07/27/2015 to 08/03/2015.

Last week there were 6 new Extensions and 21 updates. This includes themes and other extensions.

What’s going on?

I’m late, late, late and still catching up with the previous weeks. Too much things going on at the same time. New projects and maintenance of old projects :-)

One important update: there is a new way to get in contact with the team. We have created a Slack Channel where you can ask questions about Brackets and talk about features.

Have a great week!

In the meantime, you can find us on these social networks

  • Facebook
  • Google Plus
  • twitter
  • Slack Channel

Share your thoughts and stories, ask questions and get in touch with the team.

As usual, you can download the latest Brackets release. Please let us know what you think.

Updated Extensions

Name Version Description Download
Indent Guides 1.3.5 Show indent guides in the code editor.
Preferences setup UI 1.3.0 Form for editing preferences.
PHP Syntax Hint 1.1.5 PHP Syntax Hint is a plugin for Brackets, which will help you alot in writing php SYNTAX, MYSQL Queries as this will auto suggest Syntax while coding, within Brackets software. The best part of it is that if you want to write some words again and again, just go to Help > Show Extensions Folder. Once folder opens up, open folder user > brackets-php-syntax-hint. There you will get php-syntax.txt, open that file and go on adding syntax and words you require and reload Brackets to enjoy writing codes(words you have added). They will appear in the dropdown as HINT, press ENTER and it's done.
Word Count 1.0.2 Stupid simple word count for current document.
Extensions Rating 0.7.2 Shows extension's download count(daily, overall, average and per version) and GitHub stats(stars and forks). Find most usable and popular extensions for Brackets via badges system! Sort extensions and find themes!
Console Plus 1.4.8 Shows console.logs &.errors without leaving Brackets. + Toolbar Icon & Notifier, Filters by logs/errors, Errors stacks details and…. a very nice look + Update: log window object error
