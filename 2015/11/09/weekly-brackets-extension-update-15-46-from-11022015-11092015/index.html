Weekly Brackets Extension Update - 15/46

Filed Under Update

Welcome to This Week’s Brackets Extension Updates, Issue 15/46.

Added or updated @Brackets extensions in the week from 11/02/2015 to 11/09/2015.

Last week there were 5 new Extensions and 13 updates. This includes themes and other extensions.

What’s going on?

After the release, is considered the time before the next release. Not much going on this week, but there is always something to do.

If you are interested in supporting the team, feel free to review and comment on existing PR’s. If you found an issue, don’t hesitate to file it and if you already fixed one of the existing issues (you know what you need to do next ;-)).

Have a great week!

In the meantime, you can find us on these social networks

  • Facebook
  • Google Plus
  • twitter
  • Slack Channel

Share your thoughts and stories, ask questions and get in touch with the team.

As usual, you can download the latest Brackets release. Please let us know what you think.

New Extensions

Name Version Description Download
js-osFtp - Operating System FTP Client Wrapper 1.0.0 This Brackets extension uses native OS FTP clients to perform FTP. You can even FTP to a mainframe!
Hakoiko Syntax Theme 1.0.0 Hakoiko Theme for Brackets. Inspired from tumblr.com HTML Editor
Plasma Light Theme 1.0.0 Designed to be a little different against the general dark grey theme that most of us are acquainted with. Programmers used to coding in C/C++ old compilers will feel right at home with this theme.
One Dark Material Theme 1.0.0 One Drak Material Theme for Brackets
True Tomorrow - Base16 1.0.0 Completely un-edited Tomorrow from Base16. No modifications whatsoever.

Special Html Characters 1.2.4 The Inline editing context menu will get an additional option to bring up a menu of common Special HTML characters to insert of the cursors position (e.g. Copyright, Trademark, Non-breaking space).
SFtpUpload 1.3.11 Upload changed files in your project to a server via SFTP(SCP)/FTP, you have total control over which to upload through the changed files panel. HTML Wrapper 1.2.0 Format <ul>, <ol>, <nav>, <select> and <tr> children tags with a single command. NOW with improved performance.
