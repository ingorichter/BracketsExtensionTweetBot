Weekly Brackets Extension Update - 14/26

Filed Under Update

Welcome to This Week’s Brackets Extension Updates, Issue 14/26.
@Brackets Extensions who have been added or updated in the week from 06/15/2014 to 06/23/2014.

This week there were 5 new Extensions and 16 updates.

A couple of really awesome timesavers are in the list this week. Check out the great Brackets Color Palette to pick any color directly from an image and paste the selected color value into your document.
A lot of users asked for theming support in Brackets. Check out the updated version of Themes. This extension is great to customize Brackets look and feel.
BracketstoIX is the swiss army knife for all sorts of text manipulation from sorting, transformations, whitepace/tab conversion, etc. Check out the documentation.
I’m going to blog about this extension in the next couple of days.
