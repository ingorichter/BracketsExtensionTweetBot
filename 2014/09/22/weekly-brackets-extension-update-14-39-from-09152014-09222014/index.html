Weekly Brackets Extension Update - 14/39

Filed Under Update

Welcome to This Week’s Brackets Extension Updates, Issue 14/39.

Added or updated @Brackets in the week from 09/15/2014 to 09/22/2014.

This week there were 8 new Extension and 23 updates. This includes themes and other extensions.

As usual a busy week for the Brackets team. We are currently working on Release 0.44 that should be available by the end of the September. We are happy to add lots of new features, bug-fixes and improvements to this Release. We are finalizing the work on the new file tree. Since the team had some time to work with the split view feature, we are softening the edges around this great feature. We want to have it ready to give a great first impression. So stay tuned. It’ll be exciting!

If you don’t have the latest Brackets Release 0.43, get it from brackets.io. Let the team know what you think.

PHP Syntax Hint 1.0.6 PHP Syntax Hint is a plugin for Brackets, which will help you alot in writing php SYNTAX, MYSQL Queries as this will auto suggest Syntax while coding, within Brackets software. The best part of it is that if you want to write some words again and again, just go to Help > Show Extensions Folder. Once folder opens up, open folder user > brackets-php-syntax-hint. There you will get php-syntax.txt, open that file and go on adding syntax and words you require and reload Brackets to enjoy writing codes(words you have added). They will appear in the dropdown as HINT, press ENTER and that is done.
