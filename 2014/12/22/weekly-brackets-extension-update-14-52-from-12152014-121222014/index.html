Weekly Brackets Extension Update - 14/52

Filed Under Update

Welcome to This Week’s Brackets Extension Updates, Issue 14/52.

Added or updated @Brackets extensions in the week from 12/15/2014 to 12/22/2014.

This week there were 4 new Extensions and 23 updates. This includes themes and other extensions.

What’s going on

Yeah, we finally released Brackets 1.1! With the help of our great community, we were able to release Brackets before the christmas break. Have a look at the blog post that provides detailed information about the release.

For now, we are going to prepare for the 1.3 release end of January 2015. But until we get back to work around 1/5/2015, we will participate in the holiday shutdown here at Adobe. Things will slow down a bit. Let’s take some rest and look back at an awesome year for Brackets.
Hope to see everyone back beginning of next year. Refreshed, energetic and ready to make Brackets the best tool webdev’s have seen so far.

As usual, you can download the latest Brackets release and let us know what you think.
Have a great week and happy holidays!

PHP Syntax Hint 1.1.3 PHP Syntax Hint is a plugin for Brackets, which will help you alot in writing php SYNTAX, MYSQL Queries as this will auto suggest Syntax while coding, within Brackets software. The best part of it is that if you want to write some words again and again, just go to Help > Show Extensions Folder. Once folder opens up, open folder user > brackets-php-syntax-hint. There you will get php-syntax.txt, open that file and go on adding syntax and words you require and reload Brackets to enjoy writing codes(words you have added). They will appear in the dropdown as HINT, press ENTER and that is done.
