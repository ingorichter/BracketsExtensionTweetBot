Weekly Brackets Extension Update - 14/50

Filed Under Update

Welcome to This Week’s Brackets Extension Updates, Issue 14/50.

Added or updated @Brackets in the week from 12/01/2014 to 12/08/2014.

This week there were 2 new Extensions and 11 updates. This includes themes and other extensions.

What’s going on

The Brackets team is still eagerly working on the 1.1 Release. We have fixed most of the CEF issues. There is still some love for the Linux version required. It seems that we are not releasing the Linux version with the new CEF version. This is unfortunate, but you will get all the other bugfixes and improvements, except the new brackets shell. We continue to work on resolving all the outstanding issues with CEF and Linux, and hope to release it with 1.3 beginning of next year.

So, we will have some exciting times ahead. Only a handful of days before we call out the new release. In the meantime, pleae stay tuned and get the latest Brackets build and let us know what you think.
Thanks!
