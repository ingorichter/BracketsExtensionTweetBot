Weekly Brackets Extension Update - 14/43

Filed Under Update

Welcome to This Week’s Brackets Extension Updates, Issue 14/43.

Added or updated @Brackets in the week from 10/13/2014 to 10/20/2014.

This week there were 6 new Extension and 20 updates. This includes themes and other extensions.

What’s going on?

The Brackets team works hard to get ready for the 1.0 release by the end of October. The team will be in New York for the Future of Web Design Conference, where we will announce the Brackets 1.0 release.
We have released a 1.0 Beta build for everybody who wants to help testing the 1.0 release. The builds are available from here.
The forum post has some more information and the link to the community testing guide. We are happy for every test support we can get to make this a great release.
Until then, here are some numbers from the previous week.

PHP Syntax Hint 1.1.0 PHP Syntax Hint is a plugin for Brackets, which will help you alot in writing php SYNTAX, MYSQL Queries as this will auto suggest Syntax while coding, within Brackets software. The best part of it is that if you want to write some words again and again, just go to Help > Show Extensions Folder. Once folder opens up, open folder user > brackets-php-syntax-hint. There you will get php-syntax.txt, open that file and go on adding syntax and words you require and reload Brackets to enjoy writing codes(words you have added). They will appear in the dropdown as HINT, press ENTER and that is done.
