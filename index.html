Filed Under Update

Welcome to This Week’s Brackets Extension Updates, Issue 15/39.

Added or updated @Brackets extensions in the week from 09/14/2015 to 09/21/2015.

Last week there were 5 new Extensions and 8 updates. This includes themes and other extensions.

What’s going on?

The Brackets team prepares for the upcoming 1.5 release. Last minute pull request need to be based on the release branch. If you have some spare time to provide localization for your language and/or review localization PR’s, this is highly appreciated.

If you are interested in supporting the team, feel free to review and comment on existing PR’s. If you found an issue, don’t hesitate to file it and if you already fixed one of the existing issues (you know what you need to do next ;-)).

Have a great week!

In the meantime, you can find us on these social networks

  • Facebook
  • Google Plus
  • twitter
  • Slack Channel

Share your thoughts and stories, ask questions and get in touch with the team.

As usual, you can download the latest Brackets release. Please let us know what you think.
